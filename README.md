
# Linux Survival Toolkit 

## Project A: Linux System Report 

This project is a beginner Bash script that generates basic system information such as:

- Hostname 
- Current user 
- Date and time 
- Uptime 
- Disk usage 
- Memory usage 

This project helped me practice:

- Navigating Linux directories
- Writing Bash scripts 
- Debugging errors

## Project B: Log Analyzer 

This project is a beginner Bash script that analyzes a log file and provides basic insight 

The script reads a sample log file and reports:

- Total number of log entries 
- Number of ERROR messages 
- Number of WARNING messages 

This project helped me practice:

- Working with files in Bash 
- Using variables in scripts 
- Searching text with grep 
- Counting lines with wc
- Debugging syntax and command errors
