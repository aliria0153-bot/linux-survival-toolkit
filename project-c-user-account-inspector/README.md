# Project C: User Account Inspector 

This project is a beginner Bash script that inspects a Linux user account and provides detailed information about it.

## Features 

- Checks if a specific username exists on the system 
- Displays users details: 
 - Username
 - UID (User ID)
 - GID (Group ID)
 - Home directory
 - Default shell

## Usage

Run the script by providing a username as an argument:

''' bash
./user_inspector.sh <username>
